# 🤖 AI Telegram Bot

یک ربات تلگرام هوش مصنوعی با قابلیت مدیریت کاربران، حافظه شخصی، ذخیره مکالمات، کنترل دسترسی و **پنل وب مدیریت**.

---

## ✨ امکانات

### 🧠 هوش مصنوعی
- اتصال به APIهای سازگار با OpenAI
- انتخاب خودکار بهترین مدل موجود
- پشتیبانی از حافظه پویا برای کاربران
- نگهداری تاریخچه مکالمات برای پاسخ‌های هوشمندتر
- سیستم پاسخ جایگزین (Fallback) هنگام خطا

### 👥 مدیریت کاربران
- افزودن کاربران مجاز
- حذف کاربران
- مشاهده لیست کاربران
- محدودسازی دسترسی به افراد مشخص

### 🌐 پنل وب مدیریت (جدید!)
- داشبورد با آمار کلی (کاربران، پیام‌ها، زمان فعالیت)
- مشاهده لیست کاربران با قابلیت جستجو
- مشاهده حافظه پویای هر کاربر
- مشاهده تاریخچه کامل چت با طراحی حبابی (سبک تلگرام)
- طراحی ریسپانسیو برای موبایل
- لاگین امن با رمز عبور

### 📊 پنل مدیریت بات (تلگرام)
- مشاهده آمار ربات
- مشاهده تعداد پیام‌ها
- بررسی وضعیت سرویس AI
- ارسال پیام همگانی (Broadcast)

### 🛡️ امنیت
- Rate Limiting ضد اسپم
- جلوگیری از Prompt Injection
- فیلتر کلیدهای حساس حافظه
- جلوگیری از ارسال پیام‌های بیش از حد طولانی
- ثبت کامل لاگ‌ها

### 💾 ذخیره‌سازی
- SQLite Database
- ذخیره اطلاعات کاربران
- ذخیره حافظه اختصاصی هر کاربر
- ذخیره تاریخچه مکالمات

## 🚀 نصب و راه‌اندازی

### 1. کلون پروژه

```bash
git clone https://github.com/USERNAME/Ai-Telegram-Bot.git
cd Ai-Telegram-Bot
```

### 2. اجرای نصب‌کننده خودکار (روش پیشنهادی)

نصب‌کننده خودکار تمام مراحل را برای شما انجام می‌دهد:
- ساخت محیط مجازی (venv)
- نصب وابستگی‌ها
- دریافت تنظیمات از شما و ساخت فایل `.env`
- ساخت پوشه‌های مورد نیاز

**لینوکس / مک:**

```bash
chmod +x installer.sh
./installer.sh
```

**ویندوز / همه سیستم‌عامل‌ها:**

```bash
python install.py
```

> نصب‌کننده به‌صورت خط به خط از شما سوال می‌کند. فقط کافیست مقادیر را وارد کنید یا Enter بزنید تا مقدار پیش‌فرض پذیرفته شود.

### 3. نصب دستی (بدون نصب‌کننده)

اگر می‌خواهید دستی نصب کنید:

```bash
# ساخت محیط مجازی
python -m venv venv

# فعال‌سازی محیط مجازی
# لینوکس/مک:
source venv/bin/activate
# ویندوز:
venv\Scripts\activate

# نصب وابستگی‌ها
pip install -r requirements.txt
```

> شامل: `pyTelegramBotAPI`, `openai`, `python-dotenv`, `flask`

سپس فایل `.env` را دستی بسازید (نمونه در بخش زیر).

### نمونه فایل `.env` (برای نصب دستی)

> اگر از نصب‌کننده خودکار (`install.py` یا `installer.sh`) استفاده کردید، این فایل خودکار ساخته شده و نیازی به این مرحله ندارید.

```env
# توکن ربات تلگرام (از BotFather دریافت می‌کنید) - الزامی
TELEGRAM_TOKEN=YOUR_BOT_TOKEN

# کلید API هوش مصنوعی - الزامی
OPENAI_API_KEY=YOUR_API_KEY

# آدرس سرور API (پیش‌فرض: OpenAI)
OPENAI_BASE_URL=https://api.openai.com/v1

# آیدی عددی ادمین‌ها (با کاما انگلیسی جدا کنید) - بسیار مهم!
ADMIN_IDS=123456789,987654321

# رمز عبور پنل وب مدیریت
PANEL_PASSWORD=admin123

# پورت پنل وب (پیش‌فرض: 5000)
PANEL_PORT=5000

# کلید مخفی session پنل (حتماً تغییر دهید!)
PANEL_SECRET_KEY=یک-رشته-تصادفی-طولانی-اینجا-بگذارید

# --- تنظیمات اختیاری ---
# MAX_USERS=100
# RATE_LIMIT_MESSAGES=20
# RATE_LIMIT_SECONDS=60
# MAX_MESSAGE_LENGTH=4000
```

### 4. فعال‌سازی محیط مجازی و اجرا

```bash
# فعال‌سازی محیط مجازی
# لینوکس/مک:
source venv/bin/activate
# ویندوز:
venv\Scripts\activate
```

### 5. اجرای ربات تلگرام

```bash
python bot.py
```

### 6. اجرای پنل وب مدیریت

```bash
python web_panel.py
```

سپس در مرورگر خود به آدرس زیر بروید:

```
http://localhost:5000
```

> برای اجرا روی سرور: `http://YOUR_SERVER_IP:5000`

## 🌐 راهنمای پنل وب مدیریت

### صفحه ورود
- رمز عبور پیش‌فرض: `admin123` (از طریق `.env` قابل تغییر)
- پس از ۱ ساعت بدون فعالیت، نشست منقضی می‌شود

### داشبورد
- **۴ کارت آماری**: تعداد کاربران، کل پیام‌ها، تعداد مکالمات، زمان فعالیت
- **جستجو**: بر اساس نام کاربر یا آیدی تلگرام
- **جدول کاربران**: نام، آیدی، تعداد پیام، آخرین فعالیت، تاریخ عضویت
- با کلیک روی دکمه «مشاهده» وارد صفحه جزئیات کاربر می‌شوید

### صفحه جزئیات کاربر
- **کارت پروفایل**: نام، آیدی، تعداد پیام/حافظه/چت
- **حافظه کاربر**: تمام اطلاعات ذخیره‌شده توسط هوش مصنوعی (نام، شهر، شغل و...)
- **اطلاعات حساب**: تاریخ عضویت، آخرین فعالیت
- **تاریخچه چت**: پیام‌های کاربر (حباب سبز، سمت راست) و پاسخ‌های بات (حباب سفید، سمت چپ)

### نکات امنیتی پنل
- رمز عبور را در `.env` تغییر دهید
- `PANEL_SECRET_KEY` را به یک رشته تصادفی طولانی تغییر دهید
- پورت پنل را در فایروال سرور محدود کنید
- پنل فقط خواندنی است و هیچ تغییری در دیتابیس ایجاد نمی‌کند

- شهر

## 🛠️ مدیریت دیتابیس از خط فرمان

ابزار `manage_db.py` برای مدیریت دیتابیس بدون نیاز به ربات:

```bash
# نمایش راهنما
python manage_db.py

# افزودن کاربر
python manage_db.py add 123456789 Ali

# حذف کاربر
python manage_db.py remove 123456789

# مشاهده لیست کاربران
python manage_db.py list

# مشاهده حافظه کاربر
python manage_db.py memory 123456789

# ثبت/ویرایش دستی حافظه کاربر
python manage_db.py setmemory 123456789 city تهران
python manage_db.py setmemory 123456789 job برنامه‌نویس پایتون
```
